ABOUT SEEUN

안녕하세요. 웹 퍼블리셔 한세은 입니다.
단순히 웹 페이지를 구현하는 것에 그치는 것이 아니라 웹 표준, 웹 접근성,
그리고 웹 호환성을 항상 고려하여 마크업 하는
웹 퍼블리셔가 되고자 합니다.

인터렉션을 구현하여 웹 페이지의 정보를 효과적으로 전달하고
디자인의 퀄리티를 높여주며 재미 요소를 부여하는 것에 흥미를 느낍니다.
주어진 일에 최선을 다하며 새로운 것을 접하는 것에 두려워 하지 않고
항상 도전적이고 열린정신의 자세로 임할 것입니다.
모든 유저가 쉽게 이용할 수 있는 웹 환경을 구현하고 더 나아가
고객사의 프로젝트에 최선을 다할 것입니다.

01
Html5 / Css

웹 표준 및 웹 접근성을 이해하며 이에 따른 작업이 가능합니다. css 3.0요소 까지 적용 가능합니다.

02
Javascript

스크롤을 이용한 이벤트 제어와 페이지 리사이징에 반응하는 스크립트 구현이 가능합니다.

03
JQuery

Query 라이브러리와 메서드 방식을 활용하여 이벤트 구현이 가능합니다.

04
Bootstrap

다양한 구성 요소와 유틸리티 클래스를 활용하여 효율적으로 코드를 작성 할 수 있습니다.

05
Photoshop

레이어 속성을 이용한 이미지 합성 작업과 필터 기능을 사용한 보정 작업이 가능합니다.

06
Illustrator

레이어 속성과 패스를 이용한 백터 이미지 작업이 가능하고 디자인 및 시안 제작이 가능합니다.

07
Figma

ux/ui 설계와 프로토타이핑이 가능하며 Figma를 활용한 원활한 협업 경험이 있습니다.

08
D3.js

D3.js 라이브러리를 사용해 다양한 형태의 데이터 시각화된 차트를 수정 후 구현 할 수 있습니다.

09
Angular

Angular로 작성된 코드 분석하고 효율적으로 수정 및 유지보수 할 수 있습니다.
